# Spec 00 — 프로젝트 개요

## 서비스명
**Glyph Art** (글자그림)

## 한 줄 정의
이미지 속 객체를 감지해, 각 영역을 그 객체의 이름(글자)으로 채우는 Semantic Typography Art 생성기.

## 핵심 차별화
- 단순 ASCII 아트(기호 변환) X → **의미 있는 글자**로 채움
- 100% 클라이언트 처리 (사진 서버 미전송, 개인정보 안전)
- 다국어 동시 지원 (한국어 / 영어 / 일본어)

---

## 범위 (1주일 MVP)

### ✅ Must Have
- 이미지 업로드 (drag & drop + 클릭)
- YOLOv11-seg 객체 감지 + 마스크 추출
- Canvas 렌더: 마스크 영역 → 해당 클래스명 글자로 채우기
- 다국어 전환 (한/영/일)
- PNG 다운로드

### 🟡 Should Have (Day 6)
- 실시간 카메라 모드 (WebRTC)

### 🔵 Nice to Have (시간 남으면)
- 폰트/밀도/색상 옵션 커스터마이징
- SVG 다운로드

### ❌ Out of Scope (2차)
- 감정 모드 (LLM 활용)
- AI 컨텍스트 인식 (Gemini Nano)
- SAM2 마스크 정제
- 애니메이션 파티클 모드

---

## 사용자 플로우

```
1. 랜딩 화면 진입
2. 이미지 업로드 (drag & drop or 클릭 or 카메라)
3. 모델 추론 중 로딩 표시
4. 결과 Canvas 렌더링
5. 언어 전환 (한/영/일) → 즉시 재렌더
6. PNG 다운로드
```

---

## 비기능 요구사항
- 첫 추론: 모델 다운로드 포함 30초 이내 (이후 캐시)
- 재추론 (언어 전환): 1초 이내 (재추론 없이 Canvas만 재렌더)
- 지원 브라우저: Chrome 120+, Edge 120+ (WebGPU 우선, CPU fallback)
- 모바일: 기본 동작은 하되 UX 최적화는 2차
