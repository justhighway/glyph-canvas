## 컴포넌트 명세

### `components/ImageUploader.tsx`

**상태:** idle → dragover → processing

**Props:**
```ts
type Props = {
  onImageSelect: (file: File) => void
  disabled: boolean
}
```

**동작:**
- 드래그앤드롭 영역 (점선 테두리)
- 클릭 시 파일 선택 다이얼로그
- 지원 형식: jpg, png, webp
- 파일 크기 제한: 10MB
- disabled 시 UI 비활성화 (추론 중)

---

### `components/ResultViewer.tsx`

**Props:**
```ts
type Props = {
  imageFile: File
  segmentationResult: SegmentationResult | null
  language: Language
  isRendering: boolean
}
```

**동작:**
- Canvas 요소를 실제 이미지 비율로 표시
- isRendering 시 오버레이 스피너
- 결과 없으면 원본 이미지만 표시

## 상태별 UI

| 상태 | 표시 |
|------|------|
| idle | 업로드 영역 표시 |
| loading (모델) | ProgressBar + "AI 모델 로딩 중..." |
| loading (추론) | ProgressBar + "분석 중..." |
| done | 결과 Canvas + 컨트롤 패널 |
| error | 에러 메시지 + 재시도 버튼 |

## 디자인 토큰 (Tailwind 기반)

```
배경: zinc-950
카드: zinc-900
텍스트 primary: zinc-50
텍스트 secondary: zinc-400
강조: violet-500
업로드 테두리: zinc-700 (hover: violet-500)
```

다크 모드 전용 (라이트 모드 없음 — 결과 이미지가 어두운 배경에서 더 잘 보임).

## 카메라 모드 (Day 6)

### `components/CameraCapture.tsx`

```ts
type Props = {
  onFrame: (imageData: ImageData) => void
  isActive: boolean
}
```

**동작:**
- `navigator.mediaDevices.getUserMedia` 로 카메라 스트림
- `requestAnimationFrame` 루프로 매 프레임 캡처
- 추론 완료 후 다음 프레임 (추론 중 스킵)
- 실시간 Canvas 오버레이로 결과 표시
